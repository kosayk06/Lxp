# 🎓 IThub — Кураторская панель

Десктопное приложение для кураторов образовательных групп. Позволяет управлять студентами, генерировать отчёты и анализировать статистику успеваемости.

![Version](https://img.shields.io/badge/version-2.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-Free%20for%20personal%20use-orange)

---

## 🚀 Быстрый старт

1. Запустите **Ithub_analytics.exe**
2. Войдите с данными от LXP (email и пароль) (Вы обязательно должны быть Куратором)
3. Выберите группу и студентов
4. Генерируйте отчёты, статистику или аналитику

📖 **Подробная инструкция:** см. файл `Инструкция по пользованию.txt`

---

## ✨ Основные возможности

### 📋 Отчёты
- Индивидуальные отчёты для каждого студента
- Групповые отчёты в один файл
- Упрощённый режим (без контрольных точек)
- Предпросмотр и копирование в буфер обмена
- Исключение дисциплин из отчётов

### 📊 Статистика
- Средний балл и успеваемость по дисциплинам
- Процент посещаемости
- Экспорт в CSV и TXT
- Итоговая сводка по группе

### 📈 Аналитика
- Рейтинг студентов
- Выявление должников
- Корреляционный анализ посещаемости и успеваемости
- Динамика изменений (сравнение снимков)

### 🎨 Персонализация
- **Три темы:** светлая, тёмная, системная
- **Размер шрифта:** маленький, средний, большой
- **Заметки к студентам** для отслеживания работы

### 🔍 Удобство
- Быстрый поиск (Ctrl+F)
- Фильтры: Все / Активные / Архив
- Множественный выбор (Ctrl+клик)
- Контекстное меню (правый клик)
- Автоматический вход и кеширование данных

## 🐛 Решение проблем

### Не загружаются группы
- Проверьте интернет-соединение
- Проверьте правильность email и пароля
- Удалите файл `groups_cache.json` (если есть) и перезапустите

### Ошибка авторизации
- Убедитесь, что используете данные от LXP
- Очистите данные входа: Удалить файл **settings.json**

### Приложение не запускается
- Проверьте, что файл не заблокирован антивирусом
- Запустите от имени администратора

### Приложение зависает
- Дождитесь завершения текущей операции
- Перезапустите приложение
