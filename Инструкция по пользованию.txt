1. ПЕРВЫЙ ЗАПУСК И ВХОД В СИСТЕМУ
═══════════════════════════════════════════════════════════════════════════════

1.1. Запуск приложения
──────────────────────────────────────────────────────────────────────────────
• Дважды кликните по файлу "Ithub_analytics.exe"
• При первом запуске Windows может показать предупреждение безопасности
• Нажмите "Подробнее" → "Выполнить в любом случае"

1.2. Вход в систему
──────────────────────────────────────────────────────────────────────────────
• Введите ваш email от LXP (например: ivanovBB24@magas.ithub.ru)
• Введите пароль от LXP
• Нажмите кнопку "🔐 Войти" или клавишу Enter

⚠️ ВАЖНО: Используйте те же данные, что и для входа на сайт newlxp.ru!

1.3. Автоматический вход
──────────────────────────────────────────────────────────────────────────────
• После первого успешного входа данные сохраняются
• При следующем запуске вход произойдёт автоматически
• Для выхода используйте кнопку "🚪 Выйти" в правом верхнем углу

2.3. Статус-бар (внизу окна)

Показывает:
• Количество групп и студентов
• Количество заметок
• Статус текущей операции

3. ВКЛАДКА "ОТЧЁТЫ"
═══════════════════════════════════════════════════════════════════════════════

3.1. Поиск студентов
──────────────────────────────────────────────────────────────────────────────
• Введите ФИО, email или телефон в поле поиска (🔍)
• Поиск работает в реальном времени
• Для очистки поиска нажмите кнопку "✕"
• Горячая клавиша: Ctrl+F

3.2. Фильтры
──────────────────────────────────────────────────────────────────────────────
• Все — показать все группы
• Активные — только активные группы
• Архив — только архивные группы

3.3. Выбор студентов
──────────────────────────────────────────────────────────────────────────────
• Один клик — выбрать одного студента
• Ctrl+клик — выбрать несколько студентов
• Клик по группе — выбрать всю группу
• Правый клик — контекстное меню

3.4. Генерация отчётов
──────────────────────────────────────────────────────────────────────────────

А) Индивидуальные отчёты:
   1. Выберите одного или нескольких студентов
   2. Нажмите "📄 Генерировать отчёты"
   3. Подтвердите действие
   4. Дождитесь завершения (прогресс показывается внизу)
   5. Отчёты сохраняются в папку "reports"

Б) Групповой файл:
   1. Выберите группу (кликните по названию группы)
   2. Нажмите "📑 Групповой файл"
   3. Подтвердите действие
   4. Все отчёты группы сохранятся в один файл

3.5. Предпросмотр отчётов
──────────────────────────────────────────────────────────────────────────────
• После генерации автоматически откроется окно предпросмотра
• Выберите студента из выпадающего списка
• Кнопка "📋 Скопировать отчёт" — скопировать в буфер обмена
• Отключить предпросмотр можно в Настройках

3.6. Контекстное меню (правый клик)
──────────────────────────────────────────────────────────────────────────────
• 📄 Отчёт — сгенерировать отчёт для выбранного студента
• 📋 Копировать отчёт — скопировать последний отчёт
• 📝 Заметка — добавить/редактировать заметку о студенте
• 📋 Email — скопировать email студента
• 📋 Телефон — скопировать телефон студента
• 📋 ФИО — скопировать ФИО студента

3.7. Заметки к студентам
──────────────────────────────────────────────────────────────────────────────
• Правый клик на студенте → "📝 Заметка"
• Введите текст заметки (например: "Проблемы с посещаемостью")
• Нажмите "💾 Сохранить"
• Студенты с заметками помечены значком 📝
• Заметки сохраняются в файл "student_notes.json"


═══════════════════════════════════════════════════════════════════════════════
4. ВКЛАДКА "СТАТИСТИКА"
═══════════════════════════════════════════════════════════════════════════════

4.1. Загрузка статистики
──────────────────────────────────────────────────────────────────────────────
1. Выберите группу из выпадающего списка
2. Нажмите "📊 Загрузить статистику"
3. Дождитесь завершения загрузки (показывается прогресс)

4.2. Просмотр статистики
──────────────────────────────────────────────────────────────────────────────
Таблица показывает для каждой дисциплины:
• Название дисциплины
• Средний балл студентов
• Средний максимальный балл
• Процент успеваемости
• Процент посещаемости

Значки:
• 🌟 — отлично (≥90%)
• ✅ — хорошо (≥70%)
• 📘 — удовлетворительно (≥50%)
• ⚠️ — плохо (<50%)
• ❌ — нет данных

4.3. Итоговая сводка
──────────────────────────────────────────────────────────────────────────────
Внизу отображается:
• 👥 — количество студентов
• 📈 Успев — средняя успеваемость по всем дисциплинам
• 📅 Посещ — средняя посещаемость
• 📅 — текущий учебный период

4.4. Экспорт статистики
──────────────────────────────────────────────────────────────────────────────
1. Нажмите "💾 Экспорт"
2. Выберите формат:
   • Да — экспорт в CSV (для Excel)
   • Нет — экспорт в TXT (текстовый файл)
3. Файл сохранится в папку "reports"


═══════════════════════════════════════════════════════════════════════════════
5. ВКЛАДКА "АНАЛИТИКА"
═══════════════════════════════════════════════════════════════════════════════

5.1. Загрузка аналитики
──────────────────────────────────────────────────────────────────────────────
1. Выберите группу из выпадающего списка
2. Нажмите "📈 Загрузить аналитику"
3. Дождитесь завершения (может занять несколько минут)

5.2. Что показывает аналитика
──────────────────────────────────────────────────────────────────────────────
• Рейтинг студентов по успеваемости
• Список должников (студенты с долгами)
• Анализ посещаемости vs успеваемости
• Корреляционный анализ
• Динамика изменений (если есть сохранённые снимки)

5.3. Сохранение снимков
──────────────────────────────────────────────────────────────────────────────
• Нажмите "💾 Сохранить снимок" после загрузки аналитики
• Снимок сохраняет текущее состояние успеваемости
• При следующей загрузке будет показана динамика изменений
• Снимки сохраняются в файл "history.json"

💡 СОВЕТ: Сохраняйте снимки регулярно (раз в месяц), чтобы отслеживать
          прогресс студентов!

5.4. Экспорт аналитики
──────────────────────────────────────────────────────────────────────────────
1. Нажмите "📄 Экспорт"
2. Выберите формат (CSV или TXT)
3. Файл сохранится в папку "reports"


═══════════════════════════════════════════════════════════════════════════════
6. ВКЛАДКА "НАСТРОЙКИ"
═══════════════════════════════════════════════════════════════════════════════

6.1. Общие настройки
──────────────────────────────────────────────────────────────────────────────
• Упрощённые отчёты (без КТ) — отчёты без контрольных точек
• Автоматический предпросмотр отчётов — показывать окно предпросмотра

6.2. Оформление
──────────────────────────────────────────────────────────────────────────────
Тема оформления:
• ☀️ Светлая — классический светлый интерфейс
• 🌙 Тёмная — современный тёмный интерфейс
• 💻 Системная — автоматически из настроек Windows

Размер шрифта:
• Маленький — для больших экранов
• Средний — рекомендуется
• Большой — для лучшей читаемости

⚠️ После изменения размера шрифта нужно перезапустить приложение!

6.3. Учётные данные
──────────────────────────────────────────────────────────────────────────────
• Показывает ваш email
• Кнопка "🗑 Очистить вход" — удалить сохранённые данные входа
• После очистки при следующем запуске нужно будет войти заново

6.4. Исключение дисциплин
──────────────────────────────────────────────────────────────────────────────
Позволяет исключить дисциплины из отчётов и статистики:

1. Выберите группу из списка
2. Нажмите "🔄 Загрузить" (если дисциплины ещё не загружены)
3. Снимите галочки с дисциплин, которые нужно исключить
4. Изменения сохраняются автоматически

💡 СОВЕТ: Исключайте дисциплины, которые не влияют на итоговую оценку
          (например, факультативы)

8. СОВЕТЫ И РЕКОМЕНДАЦИИ
═══════════════════════════════════════════════════════════════════════════════

8.1. Оптимизация работы
──────────────────────────────────────────────────────────────────────────────
• Используйте кеш групп — данные обновляются раз в 24 часа автоматически
• Исключайте ненужные дисциплины в настройках
• Используйте упрощённые отчёты для быстрой генерации
• Сохраняйте снимки аналитики регулярно

8.2. Работа с отчётами
──────────────────────────────────────────────────────────────────────────────
• Для быстрого доступа используйте предпросмотр и копирование в буфер
• Групповые файлы удобны для отправки всей группе сразу
• Отчёты сохраняются с датой и временем в названии файла

8.3. Работа с заметками
──────────────────────────────────────────────────────────────────────────────
• Используйте заметки для отслеживания проблемных студентов
• Заметки видны только вам
• Примеры заметок:
  - "Проблемы с посещаемостью"
  - "Нужна дополнительная консультация"
  - "Отличный прогресс!"

8.4. Папка reports
──────────────────────────────────────────────────────────────────────────────
• Все отчёты, статистика и аналитика сохраняются в папку "reports"
• Папка создаётся автоматически рядом с exe-файлом
• Файлы именуются с датой и временем для удобства
• Периодически очищайте старые файлы

9. РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

9.1. Не загружаются группы
──────────────────────────────────────────────────────────────────────────────
Причины:
• Нет интернет-соединения
• Неправильный email или пароль
• Проблемы с сервером LXP

Решение:
1. Проверьте интернет-соединение
2. Проверьте правильность email и пароля
3. Попробуйте выйти и войти заново
4. Удалите файл "groups_cache.json" и перезапустите приложение

9.2. Ошибка авторизации
──────────────────────────────────────────────────────────────────────────────
Решение:
1. Убедитесь, что используете данные от LXP
2. Проверьте раскладку клавиатуры при вводе пароля
3. Очистите данные входа: Настройки → Очистить вход
4. Попробуйте войти заново

9.3. Приложение не запускается
──────────────────────────────────────────────────────────────────────────────
Решение:
1. Проверьте, что файл не заблокирован антивирусом
2. Запустите от имени администратора (правый клик → Запуск от имени администратора)
3. Проверьте, что у вас Windows 10 или 11
4. Переместите exe-файл в другую папку (не на рабочий стол)

9.4. Приложение зависает
──────────────────────────────────────────────────────────────────────────────
Причины:
• Долгая загрузка данных (это нормально для больших групп)
• Проблемы с интернетом

Решение:
1. Дождитесь завершения текущей операции (смотрите на прогресс-бар)
2. Если зависло надолго — закройте и перезапустите приложение
3. Попробуйте загрузить меньшую группу

9.5. Не генерируются отчёты
──────────────────────────────────────────────────────────────────────────────
Решение:
1. Проверьте, что студент выбран (не группа)
2. Проверьте интернет-соединение
3. Дождитесь завершения предыдущей операции
4. Проверьте логи в нижней части окна

9.6. Не сохраняются файлы
──────────────────────────────────────────────────────────────────────────────
Решение:
1. Проверьте права доступа к папке с приложением
2. Запустите приложение от имени администратора
3. Переместите exe-файл в папку с правами записи (например, Документы)

9.7. Проблемы с темой оформления
──────────────────────────────────────────────────────────────────────────────
Решение:
1. Попробуйте переключить тему в Настройках
2. Если тема "Системная" работает некорректно, выберите "Светлая" или "Тёмная"
3. Перезапустите приложение после смены темы

9.8. Файлы настроек
──────────────────────────────────────────────────────────────────────────────
Если что-то работает некорректно, можно удалить файлы настроек:
• settings.json — основные настройки
• student_notes.json — заметки о студентах
• groups_cache.json — кеш групп
• history.json — история снимков аналитики

⚠️ ВНИМАНИЕ: После удаления этих файлов все настройки и заметки будут потеряны!
